# 직거래장터 웹 지도 대시보드

주소 기준 지역 분류, 오늘 운영 장터 판정, 이번 달 운영 현황, Google Maps 연결, 지역별 현재 날씨를 제공하는 정적 웹 대시보드입니다.

## 공개 주소

GitHub Pages를 활성화하면 다음 주소로 공유할 수 있습니다.

`https://cfokk.github.io/direct-market-dashboard/`

## GitHub Pages 활성화

저장소의 `Settings → Pages`에서 다음과 같이 설정합니다.

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

## 업데이트

대시보드 파일을 수정해 `main` 브랜치에 반영하면 같은 공개 주소가 갱신됩니다.

## 외부 서비스

- 지도: OpenStreetMap / Leaflet
- 날씨: Open-Meteo
- 위치 확인: Google Maps 링크
